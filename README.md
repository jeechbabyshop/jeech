# Jeech Baby Shop - E-commerce Website

A full-stack e-commerce catalogue web app for Jeech Baby Shop in Murang'a, Kenya.

## Features

### Customer Features
- Browse baby products (clothes, shoes, blankets, toys, accessories)
- View product details with multiple images
- Filter products by category
- Search products
- Order products via WhatsApp with auto-filled message
- Contact form for inquiries
- Mobile-responsive design

### Admin Features
- Secure Firebase authentication
- Dashboard with analytics
- Full CRUD operations for products
- Multiple image upload for products
- Manage testimonials
- View and reply to customer messages
- Featured products toggle

## Tech Stack

- **Frontend**: React with Vite, Tailwind CSS, Framer Motion
- **Backend**: Firebase (Authentication, Firestore, Storage)
- **Routing**: React Router v6
- **Icons**: React Icons
- **Notifications**: React Hot Toast
